# SIFT — Signal Intelligence website

- **Root** (`index.html`, `assets/`, `hero-signal.mp4`) — production build served by GitHub Pages at https://www.sifttechnology.com/
- **`app/`** — source code (React 19 + Vite + Tailwind v4 + Motion). To rebuild: `cd app && npm install && npm run build`, then copy `dist/index.html`, `dist/assets/` and `dist/hero-signal.mp4` to the repo root.
- **`legacy/`** — previous static version of the site, kept intact (browsable at /legacy/index.html).
